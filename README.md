# 🛡️ Guardian-Nexus: Unified Security Intelligence Platform

Guardian-Nexus is a professional-grade Cyber Security suite designed to provide 360-degree protection for web applications. It addresses two critical security domains: **Internal Secure Coding** and **External Automated Threat Defense.**

## 🚀 Key Modules

### 1. 🔍 Stark-Audit (SAST Module)
An automated **Static Application Security Testing** engine that scans Python/JS source code for vulnerabilities.
*   **Credential Leak Prevention:** Identifies hardcoded passwords, API keys, and secrets.
*   **Vulnerability Detection:** Uses Regex-based pattern matching to flag SQL Injection and XSS risks.
*   **Security Compliance:** Ensures code follows OWASP Top 10 best practices.

### 🤖 2. Bot-Slayer (Behavioral Biometrics)
An invisible defense system that replaces traditional CAPTCHAs with behavioral analysis.
*   **Mouse Entropy Tracking:** Analyzes the organic randomness of human mouse movements to identify scripts.
*   **Temporal Analysis:** Detects bots that execute actions with non-human "perfect" timing.
*   **Proactive Blocking:** Automatically denies access to requests originating from automated tools.

## 🛠️ Technical Stack
*   **Backend:** Python (Flask)
*   **Frontend:** JavaScript (Event API), HTML5, CSS3
*   **Security Logic:** Regular Expressions (NLP patterns), Behavioral Modeling

2. **Bot-Slayer:** Move your mouse and click "Human" to verify. Click "Simulate Bot" to see the engine detect and block an automated request in real-time.
